# Sistem Pakar Diagnosis Stunting

Sistem pakar berbasis web untuk mendiagnosis penyakit stunting pada anak menggunakan metode Forward Chaining. Sistem ini dikembangkan menggunakan **PHP** dan **MySQL** sebagai bagian dari Tugas Akhir/Skripsi.

## Deskripsi
Aplikasi ini dirancang untuk membantu deteksi dini stunting pada anak berdasarkan gejala-gejala yang diamati. Sistem menggunakan metode Forward Chaining untuk memproses data gejala dan menghasilkan diagnosis beserta persentase tingkat kecocokan serta rekomendasi penanganan.

## Teknologi yang Digunakan
- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Metode:** Forward Chaining

## Fitur Utama
- Manajemen data gejala dan penyakit
- Konsultasi diagnosis berbasis gejala
- Hasil diagnosis dengan persentase kecocokan
- Rekomendasi solusi penanganan
- Riwayat konsultasi pengguna
